# Web

Vanilla Vite app consuming the `wuxing-wasm` package. No frameworks. Published at [https://calendar.pataruco.com](https://calendar.pataruco.com).

## Development

```sh
just web-dev
```

This builds the WASM crate first, then starts the Vite dev server.

## Production build

```sh
just web-build
```

## Preview

```sh
just web-preview
```
